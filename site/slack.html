BeerBods Slack App

Hi! I'm a Slack app that can help with BeerBods.

Installation

Click the button below to install me!

Usage

After I'm added to your Slack workspace, you should be able to type /beerbods and I'll show you what this week's beer is. It should look a bit like this:

Everyone in the channel you're in will be able to see the beer (but not the "Working on it" message).

If you need to lookup next week's beer, type /beerbods +1.
Bit behind on your beers? Type /beerbods -1 to lookup last week's beer.
You can go up to 3 weeks ahead (/beerbods +3) or up to 4 weeks back (/beerbods -4)

You can get a reminder about any of these in Slack by typing /beerbods help

Author

I was made by Josh Anderson, a fan of BeerBods.

Privacy

Text you type after /beerbods is logged anonymously so I can improve to be more helpful.
